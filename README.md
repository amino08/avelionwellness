# Avelion Wellness

Premium research peptide storefront and Avelion Care landing page.

## Production URL

https://avelionwellness.com

## Environment

Copy `.env.example` to `.env.local` for local development:

```bash
NEXT_PUBLIC_SITE_URL=https://avelionwellness.com
```

## Develop

```bash
npm install
npm run dev
```

## Production build

```bash
npm install
npm run build
npm start
```

## Deploy (Vercel)

- **Framework:** Next.js
- **Build command:** `npm run build`
- **Output:** default (Next.js)
- **Install command:** `npm install`
- **Environment variable:** `NEXT_PUBLIC_SITE_URL=https://avelionwellness.com`
